icecadiko

Listening radiko "LIVE" via icecast2

What is icecadiko?

  • radikoをicecast2経由で視聴するためのツール．
  • icecast2対応の各種クライアントでradikoを視聴可能に．
  • icecast2のURLを切り替えることで，radikoの選局も可能．
  • radikoのプレミアムにも対応．
  • Raspbianで動作確認．

Demo Video

* icecast2クライアントとしてMusic Player Daemonを，Music Player DaemonクライアントとしてiPhoneのMPoDを使用しています．
